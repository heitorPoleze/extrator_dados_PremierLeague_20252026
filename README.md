# extrator_dados_PremierLeague_20252026
Mineração de dados do campeonato de futebol inglês Premier League temporada 2025/2026. Limpeza e algumas funções de estatística para demonstrar o funcionamento correto do sistema.

Para o projeto funcionar corretamente: 
  - pip install -r requirements.txt
  - Rode o Python.

Demora um tempo para extrair tudo, mas acompanhe os avisos pelo terminal. 

O sistema está configurado para não repetir downloads de jogos que já existem na sua pasta local.

Durante o processamento, o sistema ignora automaticamente arquivos vazios ou de jogos futuros.
